# 目前实现了什么功能
### 字符串标签  生成  侧边栏DOM
1. 字符标签
``` ' #gua1/gua2/gua3/gua4# ' ```
2. 生成数组

```[ ['gua1','gua2','gua3','gua4',], ]```
4. 数组转为真正的DOM树形下拉
```
- gua1
    - gua2
        - gua3
            - gua4
```

###  DOM树形下拉做的事
1. 原生 HTML JS 实现
2. 利用DOM机制，单个实现 ```箭头``` ```内容``` 点击的判断
3. 历史记录：
   1. 上下级  箭头点击的展开
   2. 下拉激活DOM，当前激活DOM
